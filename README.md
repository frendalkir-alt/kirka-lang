# Kirka Language

Kirka — это простой интерпретируемый язык программирования с динамической типизацией, написанный на Java. Он поддерживает переменные, арифметику, логические операции, условный оператор, циклы, функции, рекурсию, ввод/вывод и комментарии. Идеально подходит для обучения основам парсинга и интерпретации, а также для создания небольших скриптов.

## Особенности

- **Типы данных**: целые числа, строки, логические (`true`/`false`).
- **Переменные**: динамическая типизация, глобальные и локальные области видимости.
- **Арифметические операции**: `+`, `-`, `*`, `/`, унарный минус.
- **Операторы сравнения**: `==`, `!=`, `<`, `>`, `<=`, `>=`.
- **Логические операторы**: `&&`, `||`, `!`.
- **Управление потоком**: `if`/`else`, цикл `while`.
- **Функции**: объявление с параметрами, возврат значения (`return`), рекурсия.
- **Ввод/вывод**: `print` (с несколькими аргументами, разделяемыми пробелом) и `input()` (с необязательным приглашением).
- **Комментарии**: однострочные `//`.
- **Расширяемость**: можно легко добавлять новые встроенные функции и конструкции.

## Требования

- Java 8 или выше (рекомендуется JDK 11+ для сборки).
- Make (опционально, для использования Makefile).

## Установка и сборка

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/kirka-lang.git
cd kirka-lang
```

### 2. Компиляция и создание JAR

```
make
```

### 3. Запуск

пока только на macos:
```
./kirka <программа .kirka>
```
